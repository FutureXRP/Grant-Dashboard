# Purpose

The federal government distributes hundreds of billions of dollars annually through grants, cooperative agreements, formula funding, and discretionary funding.

Most organizations fail to compete effectively because they pursue grants opportunistically rather than strategically.

SquareOne Compassion should instead build a **Federal Funding Portfolio** that aligns each funding opportunity with the organization's mission, existing infrastructure, and long-term strategic objectives.

This section identifies the primary federal agencies, funding programs, registration requirements, and recommended project concepts most relevant to SquareOne.

---

# Federal Funding Philosophy

Federal grants should be viewed as investments in long-term organizational capacity rather than one-time revenue sources.

Before applying, each opportunity should satisfy the following questions:

- Does this align with our strategic plan?
- Does this strengthen an existing program?
- Can we sustain the program after the grant ends?
- Can measurable outcomes be demonstrated?
- Does this create future funding opportunities?
- Does this strengthen community partnerships?

Projects that cannot answer "yes" to most of these questions should receive a lower priority.

>
> While federal grants often provide the largest awards, state funding is frequently more accessible, more relationship-driven, and better suited for expanding existing programs.

# State Funding Philosophy

State grants should accomplish four objectives:

1. Expand existing programs.
2. Strengthen community partnerships.
3. Build capacity for larger federal grants.
4. Generate measurable outcomes that improve future competitiveness.

Rather than viewing state grants as "smaller federal grants," SquareOne should treat them as strategic stepping stones that demonstrate implementation success and strengthen relationships with state agencies.

# Guiding Philosophy

2. What services did we provide?
3. What changed because of those services?

5. How can we improve next year?
